# English Master - 英语口语练习 App

## 1. 项目概述

**应用名称**: English Master  
**平台**: Android（通过 KivyMD / Buildozer 打包）  
**核心功能**: 基于间隔重复（Spaced Repetition）的英语口语翻译练习，根据学习者水平动态调整难度。  
**目标用户**: 想提高英语口语的汉语使用者，零基础到中高级均可。

## 3. 难度等级体系

| 等级 | 词汇要求 | 句子特征 | 例句 |
|------|---------|---------|------|
| 1 | 初中词汇 | 短句，主谓宾结构，无从句 | "I'm happy." |
| 2 | 高中词汇 | 简单复合句，时态变化 | "She went to the store yesterday." |
| 3 | 四级词汇 | 常见从句，介词短语 | "The book that I bought was expensive." |
| 4 | 六级/托福 | 长句，多层从句，习语 | "Even though it was raining, we decided to go anyway." |
| 5 | 雅思/GRE | 复杂结构，文化背景，俚语 | "Break a leg! You'll knock 'em dead out there." |

### 升级规则
- 连续 5 题正确 → 提示可升级
- 正确率 ≥ 80%（最近 20 题）→ 自动升级
- 连续 3 题错误 → 提示可降级

---

## 4. 间隔重复算法（SM-2 变体）

每个句子的复习周期：
- 新句子：当日复习
- 首次正确：间隔 1 天
- 第2次正确：间隔 3 天
- 第3次正确：间隔 7 天
- 第4次正确：间隔 14 天
- 第5次+正确：间隔 30 天
- 错误：重置回 1 天

## 5. 界面设计（KivyMD）

### 5.1 启动流程
```
欢迎页 → 设置目标（3步向导）→ 开始练习
```
**设置向导内容：**
1. 选择初始难度等级（1-5，或"让程序测试"）
2. 选择感兴趣的话题（可多选）
3. 设置每日目标（5/10/20/30句）

### 5.2 主练习界面
```

```

### 5.4 单词复习界面
使用闪卡（Flashcard）机制：
- 显示中文 → 回忆英文
- 显示英文 → 回忆中文
- 根据间隔重复安排复习时间

- 正确率趋势图（折线图）
- 连续学习天数
- 已掌握 / 学习中 / 未开始 句子数
- 单词库词汇量

## 6. 验收标准

1. ✅ 启动后进入设置向导，设置完成后进入练习
2. ✅ 每次显示中文句子，用户输入英文翻译
3. ✅ 提交后显示正确句子并指出差异
4. ✅ 用户可选择单词加入单词库
5. ✅ 根据记忆曲线决定下次复习时间
6. ✅ 难度等级动态调整
7. ✅ 所有数据保存在本地 JSON 文件
8. ✅ APK 可在 Android 手机上独立运行
